# SunClaw Website — Design Brainstorm

## Context
SunClaw is an AI agent distribution for the renewable energy sector, built on OpenClaw. The website needs to communicate: power, clean energy, technical credibility, and accessibility for non-technical users. It should feel like a premium developer tool meets clean energy brand.

## Idea 1: "Solar Brutalism"

**Design Movement:** Neo-Brutalist meets Clean Tech — raw, bold, unapologetic structure with warm solar energy tones.

**Core Principles:**
- Exposed structure — visible grid lines, monospaced type blocks, raw borders
- High contrast — deep charcoal backgrounds with blazing amber/gold accents
- Functional honesty — every element serves a purpose, no decorative fluff
- Industrial warmth — brutalist bones with solar warmth in color

**Color Philosophy:** The palette is built on the contrast between industrial darkness (deep slate, near-black) and the raw energy of sunlight (amber, gold, warm white). The amber represents solar power literally — it's the color of concentrated light. Against dark backgrounds, it creates a sense of energy being generated.

**Layout Paradigm:** Asymmetric grid with exposed track lines. Content blocks sit on a visible 12-column grid with deliberate misalignment. Terminal-style code blocks for the demo. Large type breaks across columns.

**Signature Elements:**
- Visible grid overlay with amber track lines
- Terminal/CLI-style interactive demo with blinking cursor
- "Energy meter" progress bars for skill categories

**Interaction Philosophy:** Interactions feel mechanical and deliberate — click states snap, hovers reveal hidden grid structure, scrolling triggers block-by-block reveals.

**Animation:** Blocks slide in from grid positions. Text types itself out. Energy meters fill on scroll. No easing — linear, mechanical motion.

**Typography System:** JetBrains Mono for headings and code. Space Grotesk for body. All caps for section labels. Extreme weight contrast (900 headings, 400 body).

## Idea 2: "Atmospheric Energy"

**Design Movement:** Environmental Minimalism — inspired by weather data visualization, atmospheric science, and the quiet power of natural forces.

**Core Principles:**
- Atmospheric depth — layered translucent surfaces like looking through atmosphere
- Data as beauty — numbers, charts, and metrics are the decoration
- Calm authority — whisper-quiet confidence, not shouting
- Natural gradients — colors shift like sky at different times of day

**Color Philosophy:** A dawn-to-dusk gradient system. The hero uses deep navy-to-amber (pre-dawn), sections transition through soft blues (morning), warm whites (midday), and golden hour tones. This isn't decorative — it maps to the solar cycle, reinforcing the product's purpose.

**Layout Paradigm:** Full-width horizontal bands that stack vertically, each with its own atmospheric "layer." Content floats within these layers with generous padding. Cards use frosted glass (backdrop-blur) to create depth. The page feels like descending through layers of atmosphere.

**Signature Elements:**
- Frosted glass cards with subtle light refraction borders
- Animated particle field in hero (representing solar photons)
- Live data ticker showing real renewable energy stats

**Interaction Philosophy:** Interactions feel weightless and natural — elements float on hover, cards tilt slightly with parallax, transitions use spring physics. Nothing feels forced.

**Animation:** Gentle parallax on scroll. Cards fade up with spring easing. Particle field responds to mouse position. Gradient backgrounds shift slowly and continuously.

**Typography System:** Instrument Sans for headings (geometric, modern, clean). Inter for body (readable, neutral). Light weights for large display text, medium for body. Generous letter-spacing on headings.

## Idea 3: "Command Center"

**Design Movement:** Mission Control Aesthetic — inspired by NASA control rooms, energy grid dashboards, and military-grade monitoring systems. Dark, precise, data-rich.

**Core Principles:**
- Operational precision — every pixel has a purpose, like a cockpit instrument
- Dark authority — dark backgrounds signal professional-grade tooling
- Live and breathing — the page feels like a live system, not a static brochure
- Information density — pack value into every viewport without clutter

**Color Philosophy:** Near-black background (#0a0f1a) with electric cyan (#00d4ff) as the primary accent — the color of energy on a dark grid map. Secondary amber (#f59e0b) for warnings/solar elements. Green (#10b981) for "online/active" states. This is the palette of control rooms and energy monitoring dashboards.

**Layout Paradigm:** Dashboard-inspired grid. The hero IS a dashboard — showing SunClaw's status, connected channels, active skills, all live. Below, content is organized in card grids that look like monitoring panels. Sidebar navigation for skills reference. The whole page feels like you're already inside the product.

**Signature Elements:**
- Animated terminal/chat demo that runs real conversation examples
- Status indicators (green dots, pulse animations) throughout the page
- Monospaced data readouts with scanning line animations

**Interaction Philosophy:** Interactions feel responsive and precise — instant hover feedback, click ripples, elements light up like activated controls. The page responds like a live system.

**Animation:** Typing animations for the chat demo. Pulse animations on status dots. Scan-line effects on data cards. Smooth counter animations for statistics. Subtle glow effects on hover.

**Typography System:** IBM Plex Mono for data/code/labels. IBM Plex Sans for body text. Small caps for section headers. Tight line-height for data, generous for prose. The monospace creates a technical, operational feel.
